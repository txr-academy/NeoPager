# NeoPager
NeoPager is a modern, LoRa-powered personal messaging device designed to provide safe, reliable, and distraction-free communication.

## Description
Unlike smartphones, NeoPager focuses on one purpose: staying connected when it matters most. Parents can send short text messages, alerts, or reminders to their child's NeoPager, and children can respond with predefined messages or simple acknowledgments. Because it uses LoRa (Long Range) communication, NeoPager offers reliable coverage across large school campuses while consuming very little power, enabling long battery life.

# Why NeoPager?

Many schools discourage or prohibit smartphones in classrooms because they can distract students and provide unrestricted internet access. At the same time, parents often want a dependable way to communicate with their children during emergencies or schedule changes.

NeoPager addresses both needs by offering a dedicated communication device that:

Has no social media, games, or web browsing.
Keeps students focused on learning.
Enables essential communication without unnecessary distractions.
Provides a simple, intuitive interface suitable for children.
